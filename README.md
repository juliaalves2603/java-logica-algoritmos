# Java - Lógica e Algoritmos

Repositório com exercícios práticos de lógica e algoritmos em Java, organizado por temas.
Objetivo: reforçar fundamentos para entrevistas e estágios em Desenvolvimento de Software.

## Conteúdo
- Variáveis e entrada de dados
- Operadores
- Condicionais (if/else)
- Laços (for/while)
- Métodos (funções)

## Como executar
1. Clone o repositório
2. Abra no IntelliJ/Eclipse ou use o terminal
3. Execute as classes dentro de `src/`

## Observações
- Exercícios com soluções comentadas
- Atualizações contínuas conforme avanço nos estudos
